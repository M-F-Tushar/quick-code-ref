## Quick Comparison

| Operation | C | C++ | Java | Python | JavaScript |
|-----------|---|-----|------|--------|------------|
| **Declaration** | `char str[]` | `string str` | `String str` | `str = "text"` | `let str` |
| **Concatenation** | `strcat()` | `+` or `append()` | `+` or `concat()` | `+` or `join()` | `+` or `concat()` |
| **Length** | `strlen()` | `.length()` | `.length()` | `len()` | `.length` |
| **Comparison** | `strcmp()` | `==` or `compare()` | `.equals()` | `==` | `===` |
| **Substring** | Manual | `.substr()` | `.substring()` | `[start:end]` | `.substring()` |
| **Search** | `strstr()` | `.find()` | `.indexOf()` | `.find()` | `.indexOf()` |
| **Replace** | Manual | `.replace()` | `.replace()` | `.replace()` | `.replace()` |
| **Upper/Lower** | `toupper()`/`tolower()` | `transform()` | `.toUpperCase()`/`.toLowerCase()` | `.upper()`/`.lower()` | `.toUpperCase()`/`.toLowerCase()` |
